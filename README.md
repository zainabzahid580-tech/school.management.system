Project Title:
School Management System (Python Tkinter + SQLite)
Description:

The School Management System is a desktop-based application developed using Python and Tkinter with SQLite as the database. It is designed to manage core school operations efficiently, including student management, teacher management, attendance tracking, fee management, and reporting.

This system implements full CRUD (Create, Read, Update, Delete) operations and includes input validation to ensure data accuracy and reliability. The application follows clean coding practices and modular design for better maintainability.
Version control is managed using Git and GitHub with feature branching. The project demonstrates proper software development workflow including commits, branching, and repository management.

**Key Features:**
User-friendly GUI built with Tkinter
Student Management (Add, Update, Delete, View)
Teacher Management System
Attendance Tracking System
Fee Management System
Reports for unpaid fees (defaulters list)
Input validation for all forms (email, phone, name, numeric fields)
SQLite database integration for data storage
Secure login system (admin authentication)
Clean and modular code structure
Git version control with feature branches
